**The Two Choices Flow Framework:**

The Two Choices Flow is a structured, calm discipline sequence:

1. **Command** - Parent gives a clear, direct command (e.g., "Please put the block in the box.")
2. **Wait** - Parent waits 5 seconds for compliance
3. **Choice** - If non-compliance, parent offers two choices: comply or face a logical consequence (e.g., "You have two choices: you can put the block in the box, or you can go to the timeout chair.")
4. **Wait** - Parent waits 5 seconds again
5. **Follow-Through** - If still non-compliant, parent calmly enforces the stated consequence

**Evaluate the parent on these 4 skills:**

1. **Effective Commands** - Did the parent give clear, direct, positively-stated commands? Were commands specific and age-appropriate? Did the parent avoid indirect commands (questions), chained commands (multiple at once), or vague commands?

2. **Wait Time** - Did the parent wait approximately 5 seconds after giving commands before repeating or escalating? Did the parent give the child time to process and respond? Did the parent avoid jumping in too quickly or repeating commands immediately?

3. **Choice Delivery** - When the child did not comply, did the parent offer clear choices with logical consequences? Were the choices calm, neutral in tone, and structured as "You can [comply] or [consequence]"? Were consequences reasonable and related?

4. **Follow-Through** - Did the parent calmly enforce the stated consequence when the child did not comply after choices were given? Was the parent consistent? Did the parent avoid empty threats, backing down, or escalating emotionally?
